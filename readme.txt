WP reCaptcha authenticate the visitor is a human verifying by Google ReCaptcha

== Description ==
WP reCaptcha add verification and protect Your form. Check Your visitor is a human or mabye spam robot.

<h4>Features</h4>
<ul>
    <li>Registration form captcha.</li>
    <li>Login form captcha.</li>
    <li>Forgot password form captcha.</li>
    <li>Comment form captcha.</li>
    <li>Own error message validate reCaptcha.</li>
</ul>

== Installation ==
<ol>
<li>Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your <code>wp-content/plugin</code> directory, or download from <a href="appsaur.co" title="Themes, plugins for Wordpress">appsaur.co</a>.</li>
<li>Activate the plugin through the WordPress admin panel.</li>
<li>Go to settings "WP reCaptcha".</li>
<li>Go to google site and generate site and secret key for "WP reCaptcha".</li>
<li>Save site and secret key in configuration "WP reCaptcha".</li>
<li>Done!</li>
</ol>
